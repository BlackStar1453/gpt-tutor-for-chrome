# 流程

## 1. 用户购买流程
1. 在扩展中查看，然后然后在网页中打开具体功能的演示
3. 展示功能。左边是action组的描述，右边是gif演示图。
4. 购买后，扩展通过监听当前tag来申请api调用并获取功能组，然后添加；调用成功后会显示通知，如果没有显示则提醒用户下载并导入。
5. 购买后可以查看我购买的功能组，然后可以查看该功能组的版本日志，然后进行更新（参考之前的更新功能）。
